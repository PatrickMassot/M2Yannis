Uses chech cohomology with compact supports for sheaves

Definition 7.1
#

Let \(M\) be a topological manifold, the rule \(\mathfrak {o}_M:K\mapsto H_{\dim M}(M,M\backslash K)\) defines a \(\mathcal{K}-\)sheaf, called the orientation \(\mathcal{K}-\)sheaf of \(M\).

If \(M\) is a manifold with boundary, let \(j:M\backslash \partial M\to M\) denote the canonical inclusion. The the orientation shaeves of \(M\) are defined as follows:

  • \(\mathfrak {o}_M:= j_*\mathfrak {o}_{M\backslash \partial M}\)

  • \(\mathfrak {o}_{M\quad rel \partial }:= j_!\mathfrak {o}_{M\backslash \partial M}\)
